# Shuang-Pro UI 设计文档

## 一、设计风格

### 1.1 设计语言

```
┌─────────────────────────────────────────────────────────────────┐
│                        设计理念                                   │
│                                                                  │
│  科技简约风 × AI 未来感                                            │
│  - 主色调：深空蓝 (#1a1a2e) + 科技紫 (#6c5ce7)                    │
│  - 辅助色：荧光绿 (#00d9ff) + 珊瑚粉 (#ff6b6b)                    │
│  - 风格关键词：专业、高效、可信赖                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 色彩规范

| 用途 | 色值 | 说明 |
|------|------|------|
| **主色** | `#1a1a2e` | 深蓝背景 |
| **主色渐变** | `linear-gradient(135deg, #1a1a2e 0%, #16213e 100%)` | 页面背景 |
| **强调色** | `#6c5ce7` | 主按钮、CTA |
| **强调色渐变** | `linear-gradient(135deg, #6c5ce7 0%, #a569cc 100%)` | 渐变按钮 |
| **高亮色** | `#00d9ff` | 链接、激活状态 |
| **成功色** | `#00b894` | 成功提示 |
| **警告色** | `#fdcb6e` | 警告提示 |
| **危险色** | `#ff6b6b` | 错误提示、删除操作 |
| **文字主色** | `#ffffff` | 主标题 |
| **文字次级** | `#b2bec3` | 副标题、描述 |
| **文字弱化** | `#636e72` | 辅助信息 |

### 1.3 字体规范

```css
/* 字体栈 */
font-family: 'Inter', -apple-system, BlinkMacSystemFont, 
             'Segoe UI', 'PingFang SC', 'Microsoft YaHei', sans-serif;

/* 字号规范 */
--text-xs: 12px;      /* 辅助文字 */
--text-sm: 14px;      /* 次要文字 */
--text-base: 16px;    /* 正文字体 */
--text-lg: 18px;      /* 小标题 */
--text-xl: 24px;      /* 模块标题 */
--text-2xl: 32px;     /* 页面标题 */
--text-3xl: 48px;     /* 大标题/Hero */
```

## 八、设计原则

1. **一致性**: 所有页面使用统一的设计语言和组件
2. **可用性**: 操作流程简单直观，3 步内完成核心功能
3. **可访问性**: 满足 WCAG 2.1 AA 标准，对比度≥4.5:1
4. **性能**: 首屏加载<3 秒，图片懒加载
5. **响应式**: 完美适配手机、平板、桌面三端
